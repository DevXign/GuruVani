A mobile-first mentorship platform connecting students with retired and senior professionals for career and exam guidance.

## Core Features (MVP)
- **Audio-first Mentorship**: Focus on high-quality voice interactions.
- **Bi-lingual Support**: Kannada and English interface.
- **Multirole System**: Separate workflows for Students, Mentors, and Admins.
- **Simulated Workflows**: 
  - Phone + OTP Login (use 1234 for OTP)
  - Mentor Application & Verification
  - Session Booking & Payment (Razorpay simulation)

## Tech Stack
- **Frontend**: Vite + React
- **Styling**: Vanilla CSS (Mobile-optimized)
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Quick Start
1. `npm install`
2. `npm run dev`

## Roles for Testing
Go to **Profile** to switch between views:
- **Student**: Default view. Search mentors and book sessions.
- **Mentor**: Manage availability and view upcoming sessions.
- **Admin**: Approve mentors and view platform stats.
